Μην χρησιμοποιείς τις προεπιλεγμένες οδηγίες του Material Design 3 (όπως απλά Card ή Surface). Το UI πρέπει να ακολουθεί αυστηρά μια σκοτεινή αισθητική Neon Glow. Το κεντρικό φόντο πρέπει να είναι πολύ σκούρο (π.χ. #0D0D12). Όλα τα στοιχεία (κουμπιά, πάνελ) πρέπει να έχουν στρογγυλεμένες γωνίες χρησιμοποιώντας RoundedCornerShape(24.dp) ή προσαρμοσμένα σχήματα. Απαγορεύεται αυστηρά η χρήση του Modifier.shadow(), διότι παράγει μόνο γκρι/μαύρες σκιές.
